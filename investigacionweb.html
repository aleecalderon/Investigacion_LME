¿Qué es JavaScript?

Introducción

JavaScript es un lenguaje de programación que se utiliza principalmente para crear contenido dinámico e interactivo en sitios web. Fue desarrollado por Netscape y su primera versión se lanzó en 1995.

Características Clave

  • Interactividad: Permite a los desarrolladores agregar funcionalidades como formularios interactivos, animaciones y actualizaciones en tiempo real.
  • Versatilidad: Puede ejecutarse en el navegador del usuario (lado del cliente) y en el servidor a través de plataformas como Node.js.
  • Integración: Se puede integrar fácilmente con HTML y CSS, los otros dos pilares del desarrollo web.
  • Event-Driven: JavaScript puede responder a eventos como clics, desplazamientos y entradas del teclado, lo que facilita la creación de aplicaciones web interactivas.

¿Cómo utilizar el JavaScript en una página web?

Tenemos entendido que HTML y CSS se encargan de generar elementos como textos y tablas, así como generar una capa estática a tu web, JavaScript se encarga de integrar elementos interactivos y animados a tu sitio web. Asimismo, este lenguaje es el encargado de controlar la relación entre las API y los sitios web que se están desarrollando.

Dado que JavaScript es un lenguaje muy parametrizable para lo que se desea agregar a una página web, se debe saber que este lenguaje de programación se usa para varios procesos y creación de elementos en una página web, en primer lugar, para mostrar para qué se usa JavaScript en una página web, tenemos que tomar en cuenta que está en la parte del cliente. Si nos alejamos un poco de la creación de elementos sobre el “display” de un sitio web, es importante mencionar que Javascript almacena una serie de API (interfaz de programación de aplicaciones) del navegador que pueden estar relacionadas con el programa de código para la creación de un sitio web.

Para utilizar JS en una página podemos tener una serie de pasos en concreto, los cuales nos explican el proceso de cómo utilizarlo en una página web:

  • Configurar el entorno de desarrollo. Se necesita de un editor de texto para escribir el código JavaScript. Se puede utilizar cualquier editor de nuestra elección, como Visual Studio Code, Sublime Text o Atom. Se tiene que tomar en cuenta que es necesario tener un navegador web para ejecutar el código JavaScript. Los navegadores más comunes, como Chrome, Firefox o Safari, son siempre excelentes opciones.
  • Crear un archivo HTML Se crea un archivo HTML en blanco y por lo general se le suele llamar “index.html”. Este archivo será la página web donde se puede ver los resultados del código JavaScript. Dentro del archivo HTML, se agrega el siguiente código básico:
    •                 
                    <!DOCTYPE html>
                       <html lang="en">
                       <head>
                          <title>Mi primera pagina web con JavaScript</title>
                       </head>
                       <body>
                          <h1>Mi primera pagina web con JavaScript</h1>
                          <script src="script.js"></script>
                       </body>
  • Crear un archivo JavaScript. Se crea un archivo JavaScript en el mismo directorio donde se encuentra el archivo HTML y por lo general le llamamos “script.js”. Este archivo contendrá el código JavaScript que se ejecutará en la página web. Este se puede enlazar con el archivo HTML agregando la etiqueta "script" en el cuerpo del documento HTML, como se muestra en el código anterior.
  • Escribir el primer programa JavaScript. Dentro del archivo “script.js”, podemos comenzar escribiendo nuestro primer programa JavaScript. Un ejemplo simple que mostraría un mensaje de saludo en la consola del navegador seria:
    •                 console.log("¡Hola, mundo!")
  • Ejecutar el código JavaScript. Abrimos el archivo HTML en nuestro navegador web. Luego, abrimos la consola del navegador (esto, haciendo un clic derecho en la página, seleccionando “Inspeccionar” y luego ir a la pestaña “Consola”). Veremos el mensaje “¡Hola, mundo!” impreso en la consola.
  • Experimentar con JavaScript Ahora que tienes un entorno básico de JavaScript configurado, se puede comenzar a experimentar con diferentes características del lenguaje, explorando variables, operadores, estructuras de control (como condicionales y bucles), funciones y mucho más.

Ventajas al utilizar JavaScript

  • Interactividad mejorada JavaScript permite crear interfaces de usuario interactivas, lo que hace que los visitantes se involucren más con tu sitio.
  • Validación del lado del cliente JavaScript puede verificar los datos de entrada antes de enviarlos al servidor, mejorando la eficiencia y la velocidad de tu sitio.
  • Mejora la carga de la página JavaScript permite cargar contenido de forma dinámica sin tener que recargar la página completa.
  • Personalización JavaScript te permite crear efectos visuales únicos y personalizados para tu sitio

Ejemplo Básico

A continuación, se muestra un ejemplo sencillo de código JavaScript que muestra un mensaje en la pantalla cuando se hace clic en un botón:

<html>
     <head>
        <title>Ejemplo de JavaScript</title>
     </head>
     <body>
        <button onclick="mostrarMensaje()">Haz clic aquí</button>

        <script>
            function mostrarMensaje() {
                alert('¡Hola, mundo!');
            }
        </script>
     </body>
     </html>
